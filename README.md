# Global Hack - MSJ Digital Studio

Este repositorio contiene el proyecto web de **Global Seguros** dentro de la carpeta `Proyecto Final`.

## Estructura del proyecto

- `Proyecto Final/index.html` - Página principal del sitio.
- `Proyecto Final/styles.css` - Estilos de la página.

## Cómo abrir el proyecto

1. Abre la carpeta `Proyecto Final` en tu editor.
2. Abre `index.html` en el navegador.

> Si usas VS Code, puedes instalar la extensión Live Server y abrir `index.html` con Live Server para ver los cambios en tiempo real.
